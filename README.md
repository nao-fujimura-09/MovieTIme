# MovieTIme

### サイトテーマ
映画好きな人の為のレビューサイト

### テーマを選んだ理由
私自身、映画鑑賞が趣味なこともあり、映画レビューサイトをよく利用するので、今回作成に至りました。
この機能があればもっといいのにと思った機能を追加しております。

### ターゲットユーザ
１０代〜６０代の老若男女

### 主な利用シーン
おうち時間の映画鑑賞前やその後に
